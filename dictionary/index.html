What is MintLang?

MintLang is an language made by MintBread.

No, this is not an programming language.

PRONOUNS

  • mi => he
  • nt => she
  • mt => it
  • ni => we
  • mimi => you
  • mint => me/I
  • ntnt => them
  • nint => they

ABBREVIATIONS

SPECIAL RULE: You can actually stack abbreviations! e.g. mt'mi'nt => it'sn't

  • 'nt => n't
  • 'mi => 's
  • 'ni => 're
  • 'wae => 'm/li>
